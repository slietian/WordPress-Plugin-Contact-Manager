# Perform ‚Äì Performance Optimization for WordPress

Perform is a lightweight and developer-friendly WordPress performance optimization plugin designed to remove unnecessary assets, scripts, and WordPress bloat to significantly improve site speed and Core Web Vitals.

It provides fine-grained control over asset loading while maintaining compatibility with popular themes, page builders, and caching solutions.

---

## üöÄ Features

### Asset Management
- Disable unused CSS and JavaScript files
- Page-level asset control for posts and pages
- Compatible with Elementor, Divi, and other major page builders

### Performance Optimization
- Remove jQuery Migrate
- Disable emojis, embeds, and XML-RPC
- Strip query strings from static resources
- Add DNS Prefetch and Preconnect
- Cache navigation menus for faster interactions

### WordPress Cleanup
- Disable unnecessary WordPress core features
- Control Heartbeat API frequency
- Manage post revisions
- Optimize WooCommerce scripts and cart fragments
- Remove WordPress version exposure

### Advanced Tools
- SSL redirection manager
- Native CDN support
- Asset preloading
- Optimized WooCommerce behavior

## üîß Installation

### Requirements
- WordPress 4.8 or higher
- PHP 7.4 or higher
- MySQL 5.5 or higher

### Steps
1. Download or clone this repository.
2. Upload the folder to `/wp-content/plugins/`.
3. Activate **Perform** from the WordPress admin dashboard.
